About Me


I am Animesh Trivedi, a researcher in the Cloud and Computing Infrastructure group at the IBM Research Lab, Zurich. Earlier in 2016, I graduated with a PhD from ETH, Zurich.

My primary research interests are in the area of operating systems, high-performance networks and storage architectures, and distributed systems. Currently I am investigating how to design high-performance analytics stack by leveraging high-performance network (40, 100 Gbps and beyond) and storage hardware (NVM flash, PCM). I am experimenting in Apache data processing ecosystem (Spark, Hadoop, Flink, etc.) with RDMA networking and flash devices. For more details see the projects section.

Projects


  • Spark-IO: Spark-IO contains various I/O acceleration plugins for Spark tailored to high-performance network and storage hardware (RDMA, NVMef, etc.). Spark-IO is based on Crail, a fast multi-tiered distributed storage system. Spark-IO currently contains two IO plugins: a shuffle engine and a broadcast module. Both plugins inherit all the benefits of Crail such as very high performance (throughput and latency) and multi-tiering (e.g., DRAM and flash). More details can be found at https://github.com/zrlio/spark-io


  • Crail Distributed File System: Crail is a fast multi-tiered distributed file system designed from ground up for high-performance network and storage hardware. It provides an asynchronous file system API leveraging the asynchronous nature of RDMA-based networking hardware and supports ultra-low latencies for both meta data and data operations. Crail is implemented in Java offering a Java API which integrates directly with the Java off-heap memory. Crail is designed for performance critical temporary data within a scope of a rack or two. More details can be found at https://github.com/zrlio/crail


  • DaRPC: Data Center Remote Procedure Call (DaRPC): DaRPC is a Java library that provides ultra-low latency RPC for RDMA capable network interfaces. More details can be found at https://github.com/zrlio/darpc


  • DiSNI: Direct Storage and Networking Interface (DiSNI): DiSNI is a Java library for direct storage and networking access from userpace. It currently provides an RDMA interface. More details can be found at https://github.com/zrlio/disni
